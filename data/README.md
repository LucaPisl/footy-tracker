# Data (not tracked)

This folder is git-ignored except for this README. Put datasets and sample videos here.

Recommended layout:
- `data/soccernet_tracking/`
- `data/sn_jersey/`
- `data/videos/`

Suggested usage:
1) Download SoccerNet-Tracking and extract into `data/soccernet_tracking/`.
2) Download sn-jersey crops and extract into `data/sn_jersey/`.
3) Place a sample HD match video at `data/videos/sample.mp4`.

Update `configs/paths_local.yaml` if your paths differ.
